img-placeholder
-------------
image placeholders completions for html, css and markdown

##### support
support html tag attibute
```html
src=""
srcset=""
poster=""
```
support css value
```css
  url()
```

support markdown value
```
  ![Alt Text](url)
```
##### config

Sublime Text -> Preferences -> Package Settings -> img-placeholder

* `domains` - sites hosting the placeholder
* `width` - width
* `height` - height
* ...
